# RoverSign

## 部署方式

```shell
├── gsuid_core
│   ├── gsuid_core
│   │   ├── plugins
│   │   │   ├── XutheringWavesUID
│   │   │   ├── RoverSign


cd gsuid_core/gsuid_core/plugins
git clone --depth=1 https://github.com/luofengliuyun/RoverSign.git
```

依赖：XutheringWavesUID 及其数据库

考虑到库街区不让重复登录，希望后续Nami也能支持。
